### Introduction

This is the server side logic of code_dweller agent that runs on your server and helps you identify and debug runtime errors and bugs in real time.

**What is it?**
code_dweller is an agentic environment that runs completely on our server and talks to your code base via our sdk.
The environment is secure, we do not touch your-code without permission, we help you track down, issues, bugs, crashes in production when you need help the most!

**How it works**

- The sdk helps the dweller_agent read the necessary information about the architecture, file structure and common errors in the code_base and the agent decides what to ask for based on the requirements
